# todo-app and JWT token

## setup dependencies
- [golang-migrate](https://github.com/golang-migrate/migrate) Install migrate CLI

- Migration
    ```
    make migrate-up
    ```

## Run API service
 - Installl dependencies
    ```
    go mod tidy && go mod vendor
    ```
 - Run server
    ```
    make run
    ```
